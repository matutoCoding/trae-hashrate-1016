## 1. 产品概述
舞台转台多层联动同步转速编排与碰撞校验生产力系统，面向舞台机械控制领域，解决多环转台联动编排、碰撞风险识别、同步精度监控等核心痛点，为舞美团队提供从参数录入到演出复演的全流程数字化支撑。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 机械工程师 | 系统管理员分配 | 录入转台参数、校验扭矩与安全范围 |
| 舞美编排师 | 系统管理员分配 | 编排运动时序、场景拆分、模板管理 |
| 演出操作员 | 系统管理员分配 | 执行演出脚本、监控同步状态、告警响应 |

### 2.2 功能模块
1. **转台录入页**：转台环参数录入、电机参数录入、升降台参数录入、参数预览与校验
2. **运动编排页**：内外环转速方向编排、启停时刻编排、加减速曲线设置、升降台叠加运动、分场景时间轴、运动脚本记录
3. **碰撞校验页**：多环联动线速度计算、反向旋转碰撞轨迹识别与标红、扭矩安全范围校验、演员站立安全校验、复合轨迹模拟
4. **同步监控页**：实时同步误差监控、卡顿/抖动告警、误差阈值设置、演出脚本执行与复演
5. **模板库页**：常用调度模板存储、模板分类与搜索、模板导入导出、模板预览与编辑

## 3. 核心流程

用户首先在转台录入页录入转台各环的物理参数和电机参数，然后进入运动编排页按演出需求编排各环的转速、方向、启停时刻及加减速曲线，并可叠加升降台运动；编排完成后进入碰撞校验页，系统自动计算线速度、识别碰撞轨迹并标红、校验扭矩与安全范围、模拟复合轨迹；校验通过后在同步监控页加载脚本执行，实时监控同步误差并对超阈值告警；常用编排可存入模板库供复用，整场运动脚本可记录供后续复演。

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深灰/炭黑背景 (#0D1117)，搭配钢青色 (#00D4AA) 作为主强调色，琥珀橙 (#FF8C00) 作为告警色，红色 (#FF3B3B) 作为碰撞/危险色
- **按钮风格**：圆角矩形，轻微3D凸起效果，主按钮填充钢青色，危险按钮填充红色
- **字体**：标题使用 Rajdhani（工业/科技感），正文使用 Source Sans 3，数据/数值使用 JetBrains Mono 等宽字体
- **布局风格**：左侧固定导航栏 + 右侧内容区，顶栏显示系统状态，卡片式模块布局
- **图标风格**：线性图标，2px描边，与 lucide-react 图标库一致
- **整体风格**：工业控制台风格，深色主题，高对比度，数据密集但层次分明

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 转台录入页 | 参数表单 | 深色卡片表单，数字输入框带单位标签，JetBrains Mono数值显示 |
| 转台录入页 | 环形预览 | Canvas同心环示意图，各环颜色编码，悬停显示参数 |
| 运动编排页 | 时间轴 | 多轨水平时间轴，每环一轨，色块表示运动段，可拖拽调整 |
| 运动编排页 | 场景列表 | 左侧垂直场景标签列表，点击切换当前编辑场景 |
| 运动编排页 | 加减速曲线图 | 小型折线图显示速度-时间曲线，支持梯形/S形切换 |
| 碰撞校验页 | 碰撞可视化 | Canvas动画显示转台俯视图，碰撞区域红色高亮闪烁 |
| 碰撞校验页 | 校验结果表 | 表格显示各项校验结果，通过绿色✓/不通过红色✗ |
| 同步监控页 | 仪表盘 | 圆形仪表盘显示各环同步误差，指针式+数字双重显示 |
| 同步监控页 | 告警横幅 | 顶部横幅式告警，琥珀橙底色，支持消音按钮 |
| 模板库页 | 模板卡片 | 网格布局卡片，缩略时间轴预览，标签分类 |

### 4.3 响应式
- 桌面优先设计，最小宽度1280px
- 1920x1080为最佳体验分辨率
- 侧边栏可折叠以适配较小屏幕

### 4.4 3D场景指引（不适用）
- 本项目以2D Canvas可视化和数据表为主，不需要3D场景
